###SOCKS5
首先到[http://ss5.sourceforge.net/](http://ss5.sourceforge.net/)去下载最新的版本
目前是: ss5-3.8.9-8.tar.gz
```bash
tar -zxf ss5-3.8.9-8.tar.gz
```
```bash
cd ss5-3.8.9
```
```bash
./configure
```
